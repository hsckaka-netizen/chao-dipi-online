# 炒地皮在线房间

第一阶段目标：先跑通真正多人在线房间的基础能力。

已实现：

- 创建房间并生成房间号。
- 玩家输入名字加入房间。
- 第一版限制 5-7 人开始。
- 房主开始后，系统按人数生成对应副数的完整扑克。
- 每人发 53 张牌。
- 底牌数量等于人数。
- 每个玩家页面只收到自己的手牌，其他玩家只显示手牌数量。
- 房间状态通过浏览器实时事件同步。
- 房主可以一键补齐到 5 个测试玩家，方便本地验证发牌。
- 发牌后进入叫主/抢主，支持选择同花色的 2 叫主或抢主。
- 如果无人叫主，房主可以触发随机指定主。
- 每次亮主、抢主和炒底后，页面会展示当前主牌花色。
- 主拿底后需要选择同数量牌贴底。
- 主拿底或玩家炒底拿底后，新进入手牌的底牌会保持选中，方便识别。
- 贴底后会停在炒底阶段，玩家可以用更大的 2 炒底，也可以选择不炒。
- 叫主/抢主和炒底阶段会显示每个玩家的等待、过、当前操作状态。
- 房主可以在牌桌状态面板中手动推进测试玩家准备流程，但不会直接隐藏真人需要处理的炒底阶段。
- 炒底结束后按最后一次亮出的 2 确定主牌。
- 主可以选择 1 张非比牌作为狗腿牌，然后进入打牌。
- 玩家列表会标注主、已暴露的狗腿和闲家；狗腿牌被打出后会自动识别狗腿。
- 右侧玩家信息会单独显示得分、被拖红五、被拖方五。
- 发牌后显示所有玩家、当前轮出牌、历史出牌。
- 玩家可以从自己的手牌中选择牌并出牌。
- 准备阶段参谋 3 暂不进入比牌，仍按花色放置。
- 打牌阶段，主花色普通牌、主花色 3、同颜色另一花色 3 会和比牌放在同一个“主牌/比牌”分组，并按主牌牌力排序。
- 手牌支持折行显示，主牌/比牌很多时不会只依赖横向滚动。
- 手牌支持点击选择、按住滑动选择多张、点击空白处取消选择。
- 首家出牌已校验基础牌型：单张、同点数多张、同花色连续多张拖拉机。
- 后手跟牌不限制牌型，只校验必须同张数，且优先跟首家花色。
- 本轮结束后会按同牌型、同首家花色的牌比较大小，胜者获得本轮 5/10/K 分并成为下轮首家。
- 比牌/主牌拖拉机支持按主牌牌力相邻组合，例如红五+方五、大小王、主牌 A+副 2。
- 真人玩家出牌后，后续轮到的虚拟测试玩家会自动出牌，直到轮到下一个真人。
- 如果一轮最大者是虚拟测试玩家，它会自动开启下一轮并继续出牌，直到轮到真人。
- 房主可以触发“测试玩家自动出牌”，但不会替真人玩家出牌。
- 准备流程里，虚拟测试玩家会自动过叫主/炒底；如果虚拟测试玩家被随机指定为主，也会自动贴底和选狗腿牌。
- 最后一轮打完后会自动进入结算，展示闲家目标分、保底、拖五、闲家/庄家每人积分和玩家明细。
- 打牌阶段会固定展示狗腿牌，并在玩家打出狗腿牌、身份变成狗腿时提示所有玩家。

## 本地启动

```bash
cd tools/chao-dipi-online
npm start
```

打开：

```text
http://localhost:3000
```

同一台电脑上可以用多个浏览器窗口模拟多个玩家。
